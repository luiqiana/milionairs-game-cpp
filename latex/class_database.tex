\doxysubsection{Detailed Description}
A Singleton class responsible for handling the My\+SQL database connection. 


\begin{DoxyItemize}
\item This class establishes a session with the database using the mysqlx library. The Singleton pattern ensures that only one database connection instance exists throughout the application\textquotesingle{}s lifecycle, optimizing resource usage. 
\end{DoxyItemize}
