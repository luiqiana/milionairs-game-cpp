\doxysubsection{Detailed Description}
A static utility class for executing raw SQL queries. 


\begin{DoxyItemize}
\item This class provides a simplified interface for standard CRUD operations (SELECT, INSERT, UPDATE) using the mysqlx library. 
\end{DoxyItemize}
